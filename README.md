# Alua Achievement Tracker
A simple cross-platform achievement tracking app, written in C# using the UNO Framework.

# Installing
Releases > Select your platform of choice.

# Building
Clone the repo with --recurse-submodules to also get Sachya, which is a backend
Build with your IDE of choice or dotnet build.
